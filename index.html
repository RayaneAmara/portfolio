Mes projets

Un projet = un travail concret que j'ai réalisé en cours. Voici les principaux, avec à chaque fois ce que j'ai fait et les compétences mises en œuvre.

Projet 1
Python SQL PowerShell MariaDB

Créer automatiquement des comptes et des bases de données

En clair : dans une entreprise, chaque employé a besoin de son propre espace pour ranger ses données. Au lieu de le faire à la main pour chaque personne (très long !), j'ai écrit un programme qui le fait tout seul.

Projet réalisé en cours (BTS SIO), à partir d'un cas d'école : l'entreprise STESIO. J'ai utilisé PowerShell pour récupérer la liste des employés, puis un script Python pour générer leurs identifiants, des mots de passe solides et le code SQL qui crée chaque base de données MariaDB avec les bons droits d'accès.

Voir les compétences que j'ai utilisées
  • Comprendre le besoin d'une entreprise
  • Créer un programme en Python
  • Automatiser des tâches répétitives
  • Gérer les droits d'accès d'une base de données
  • Tester et écrire la documentation
Projet 2
Docker Docker Compose Git Linux

Mettre une application dans une « boîte » avec Docker

En clair : une application ne marche pas toujours pareil d'un ordinateur à l'autre. Avec Docker, on la range dans une « boîte » (un conteneur) qui contient tout ce qu'il faut. Comme ça, elle marche partout, pareil.

Réalisé avec Jules Dupin. Nous avons écrit un Dockerfile (la recette de la boîte), puis utilisé Docker Compose pour lancer en même temps l'application et sa base de données, chacune dans sa propre boîte.

Voir les compétences que j'ai utilisées
  • Mettre une application dans un conteneur Docker
  • Écrire un Dockerfile (la « recette »)
  • Lancer plusieurs services avec Docker Compose
  • Travailler à deux avec Git

Arduino Électronique Embarqué

Électronique embarquée — Projets techniques Arduino

En clair : l'Arduino est une petite carte électronique programmable. J'ai conçu et assemblé des projets concrets : capteurs, actionneurs, prototypes fonctionnels.

Réalisation et maquettage de projets techniques autour de l'écosystème Arduino : montage de circuits, programmation en C embarqué, tests et validation des prototypes. Approche orientée ingénierie et résolution de problèmes matériels.

Voir les compétences mobilisées

Projet 4
VMware Cisco Linux DNS HTTPS / SSL

Projet GSB Archi-Site — Architecture réseau sécurisée multi-serveurs

En clair : un laboratoire voulait un site interne pour gérer les comptes-rendus et frais de ses visiteurs médicaux. Avec mon équipe, nous avons construit toute l'infrastructure réseau qui fait tourner ce site de façon sécurisée.

Projet de groupe (à 3 personnes) réalisé en BTS SIO SISR. Mise en place d'une architecture répartie sous VMware : serveur Web en DMZ (HTTPS/SSL), serveur de base de données, serveur DNS et serveur de sauvegarde, le tout segmenté en réseaux PROD / DMZ derrière un pare-feu, avec routeurs et switch Cisco et un service FTP interne authentifié pour les développeurs.

Voir les compétences que j'ai utilisées
  • Concevoir une architecture réseau segmentée (PROD / DMZ)
  • Virtualiser des serveurs Linux & Windows avec VMware
  • Configurer DNS, serveur Web sécurisé (HTTPS/SSL) et FTP
  • Mettre en place une base de données et des sauvegardes
  • Configurer des routeurs et switch Cisco
  • Travailler en équipe et rédiger la documentation technique
